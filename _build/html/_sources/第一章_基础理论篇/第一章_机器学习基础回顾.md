## 1.1 机器学习基本概念

### 1.1.1 什么是机器学习

机器学习是人工智能的一个重要分支，它关注如何通过算法从数据中学习规律，从而对未知数据做出预测或决策。

**核心思想**：
- 不再依赖明确的规则编程，而是通过数据训练模型
- 模型从数据中自动学习特征和模式
- 通过泛化能力对新数据做出合理预测

**三大学习范式**：
1. **监督学习**：使用带标签的数据进行学习
2. **无监督学习**：使用无标签数据进行模式发现
3. **强化学习**：通过与环境交互获得奖励/惩罚来学习策略

### 1.1.2 监督学习

监督学习是机器学习中最常用的范式，它使用有标签的数据集来训练模型。

**主要任务类型**：
- **分类问题**：预测离散标签
  - 二分类
  - 多分类
- **回归问题**：预测连续数值
  - 线性回归
  - 逻辑回归

**监督学习流程**：
```
1. 数据收集：获取带标签的训练数据
2. 特征工程：提取和选择特征
3. 模型选择：选择合适的算法
4. 模型训练：使用训练数据拟合模型
5. 模型评估：在测试数据上评估性能
6. 调优优化：调整参数提升性能
```

**常见算法**：
- 线性回归
- 逻辑回归
- 决策树
- 随机森林
- 支持向量机 (SVM)
- 神经网络
- 梯度提升树 (GBDT)
- Transformer

### 1.1.3 无监督学习

无监督学习在没有任何标签的情况下从数据中发现模式和结构。

**主要任务类型**：
- **聚类**：将相似的数据分组
  - K-means聚类
  - 层次聚类
  - DBSCAN
- **降维**：简化数据表示
  - PCA
  - t-SNE
  - LDA
- **关联规则**：发现变量间的关联
  - Apriori算法
  - FP-Growth算法

### 1.1.4 强化学习

强化学习关注智能体如何通过与环境交互来学习最优策略。

**核心概念**：
- **智能体**：执行动作的实体
- **环境**：智能体所处的场景
- **状态**：环境的当前情况
- **动作**：智能体可以采取的行为
- **奖励**：环境给出的反馈信号
- **策略**：智能体选择动作的规则

**学习过程**：
```
1. 智能体在环境中观察状态
2. 根据策略选择动作
3. 环境给出新状态和奖励
4. 根据奖励调整策略
5. 重复上述过程
```

**常见算法**：
- Q-Learning
- DQN (深度Q网络)
- PPO (近端策略优化)
- A3C (异步优势演员-评论家)
- SAC (软演员-评论家)

## 1.2 深度学习回顾

### 1.2.1 神经网络基础

神经网络受到人脑神经元的启发，由大量简单的计算单元（神经元）相互连接构成。

**神经网络组成**：
- **输入层**：接收原始输入数据
- **隐藏层**：提取特征的中间层
- **输出层**：产生预测结果

**神经元模型**：
```
输入 x₁, x₂, ..., xₙ
权重 w₁, w₂, ..., wₙ
偏置 b
激活函数 f(·)
输出 y = f(Σ(wᵢ · xᵢ + b)
```

**前向传播**：
```
输入层 → 隐藏层 → ... → 输出层
```

**反向传播**：
```
计算损失函数值
计算梯度（使用链式法则）
更新权重和偏置
```

### 1.2.2 反向传播算法

反向传播算法是训练神经网络的核心方法，它通过梯度下降优化网络参数。

**算法步骤**：
1. **前向计算**：计算每一层的输出
2. **计算损失**：比较预测值与真实值
3. **反向传播**：计算梯度
4. **参数更新**：使用梯度更新权重

**数学原理**：
```
损失函数 L = L(y, ŷ)

梯度：∂L/∂w = ∂L/∂ŷ · ∂ŷ/∂h · ∂h/∂w

权重更新：w ← w - η · ∂L/∂w
```

### 1.2.3 激活函数

激活函数为神经网络引入非线性，使其能够学习复杂的模式。

**常见激活函数**：

1. **Sigmoid函数**
   ```
   σ(x) = 1 / (1 + e^(-x))
   特点：输出在(0, 1)之间，平滑可微
   问题：梯度消失，输出不以0为中心
   ```

2. **Tanh函数**
   ```
   tanh(x) = (e^x - e^(-x)) / (e^x + e^(-x))
   特点：输出在(-1, 1)之间，以0为中心
   问题：仍有梯度消失问题
   ```

3. **ReLU函数** (推荐使用)
   ```
   ReLU(x) = max(0, x)
   特点：计算简单，缓解梯度消失，稀疏激活
   问题：死亡ReLU问题
   ```

4. **Leaky ReLU**
   ```
   Leaky ReLU(x) = max(αx, x), α > 0
   特点：解决死亡ReLU问题
   ```

5. **GELU** (Transformer中使用)
   ```
   GELU(x) = x · Φ(x)
   Φ(x)：标准正态分布的累积分布函数
   特点：更平滑，表现更好
   ```

### 1.2.4 损失函数

损失函数衡量模型预测与真实值之间的差异。

**分类任务损失函数**：

1. **交叉熵损失**
   ```
   L = - Σ y_i · log(ŷ_i)
   y_i：真实标签（0或1）
   ŷ_i：预测概率
   ```

2. **二元交叉熵**
   ```
   L = -[y·log(ŷ) + (1-y)·log(1-ŷ)]
   用于二分类
   ```

3. **多元交叉熵**
   ```
   L = - Σ C Σ y_{c,i} · log(ŷ_{c,i})
   用于多分类
   ```

**回归任务损失函数**：

1. **均方误差 (MSE)**
   ```
   L = Σ (y - ŷ)² / n
   ```

2. **平均绝对误差 (MAE)**
   ```
   L = Σ |y - ŷ| / n
   对异常值更鲁棒
   ```

3. **Huber损失**
   ```
   L = { 0.5·(y-ŷ)², if |y-ŷ| ≤ δ
        δ·(|y-ŷ| - 0.5·δ), otherwise
   ```
   结合MSE和MAE的优点

### 1.2.5 优化算法

优化算法用于更新模型参数以最小化损失函数。

**梯度下降法**：
```
w ← w - η · ∇L(w)
η：学习率
∇L(w)：损失函数的梯度
```

**常见优化算法**：

1. **SGD** (随机梯度下降)
   - 每次只用一个样本或一个mini-batch更新
   - 简单但可能震荡

2. **Momentum** (动量)
   - 引入动量项，加速收敛，减少震荡
   ```
   v ← β·v + η·∇L(w)
   w ← w - v
   ```

3. **Adam** (自适应矩估计)
   - 结合动量和自适应学习率
   - 当前最常用的优化器
   ```
   m ← β₁·m + (1-β₁)·∇L(w)
   v ← β₂·v + (1-β₂)·(∇L(w))²
   w ← w - η · m̂ / (√v̂ + ε)
   ```

4. **AdamW**
   - Adam的改进，分离权重衰减

5. **LAMB**
   - 适用于大batch size训练

## 1.3 数学基础

### 1.3.1 线性代数基础

线性代数是深度学习的基础，广泛应用于矩阵运算、特征提取等。

**核心概念**：

1. **矩阵乘法**
   ```
   C = A × B
   C[i,j] = Σ A[i,k] · B[k,j]
   ```

2. **向量点积**
   ```
   u · v = Σ u_i · v_i
   ```

3. **矩阵求导**
   ```
   ∂(A·x)/∂x = Aᵀ
   ∂(xᵀ·A·x)/∂x = (A + Aᵀ)·x
   ```

4. **特征值与特征向量**
   ```
   A·v = λ·v
   λ：特征值
   v：特征向量
   ```

**重要性质**：
- 矩阵乘法结合律
- 转置性质：(A·B)ᵀ = Bᵀ·Aᵀ
- 迹的性质：tr(A·B) = tr(B·A)
- 秩的性质：rank(A·B) ≤ min(rank(A), rank(B))

### 1.3.2 微积分基础

微积分用于计算梯度，指导模型训练。

**核心概念**：

1. **梯度**
   ```
   ∇f(x) = [∂f/∂x₁, ∂f/∂x₂, ..., ∂f/∂xₙ]ᵀ
   梯度指向函数增长最快的方向
   ```

2. **链式法则**
   ```
   d/dx f(g(x)) = f'(g(x)) · g'(x)
   ```

3. **雅可比矩阵**
   ```
   J = ∂F/∂X
   多元函数的梯度向量
   ```

4. **海森矩阵**
   ```
   H = ∂²f/∂x²
   二阶导数矩阵，描述函数曲率
   ```

**重要公式**：
- 乘积法则：d(u·v) = u·dv + v·du
- 商法则：d(u/v) = (v·du - u·dv)/v²
- 幂法则：d(xⁿ) = n·xⁿ⁻¹·dx

### 1.3.3 概率论与统计

概率论和统计学为机器学习提供理论基础。

**核心概念**：

1. **概率分布**

   - **正态分布**
     ```
     N(μ, σ²): 概率密度函数
     μ：均值，σ：标准差
     ```

   - **伯努利分布**：二值随机变量
   - **二项分布**：n次独立伯努利试验
   - **多项分布**：多分类结果

2. **期望与方差**
   ```
   E[X] = Σ x·P(X=x)  期望
   Var(X) = E[(X-E[X])²]  方差
   ```

3. **联合概率与条件概率**
   ```
   P(A,B) = P(A)·P(B|A)  联合概率
   P(B|A) = P(A,B) / P(A)  条件概率
   ```

4. **最大似然估计 (MLE)**
   ```
   θ̂ = argmax θ Σ log P(x_i | θ)
   ```

5. **贝叶斯定理**
   ```
   P(A|B) = P(B|A)·P(A) / P(B)
   ```

**重要概念**：
- 贝叶斯推断
- 先验、似然、后验
- 帕斯卡法则
- 概率链规则

## 1.4 NLP基础概念

### 1.4.1 文本表示

计算机无法直接理解文本，需要将文本转换为数值表示。

**1. 词袋模型 (Bag of Words)**
```
将文本表示为单词的频率向量
忽略词序和语法结构
```

**2. TF-IDF**
```
TF-IDF = TF · IDF
TF：词频
IDF：逆文档频率，衡量词的重要性
```

**3. Word2Vec**
- **CBOW模型**：根据上下文预测中心词
- **Skip-gram模型**：根据中心词预测上下文
- 特点：学习词的分布式表示，捕捉词义相似性

**4. GloVe**
- 全局向量模型
- 基于共现矩阵分解
- 结合全局统计和局部上下文

**5. 嵌入层**
- 神经网络中的词汇表映射
- 可学习的词向量
- 参数共享，减少参数量

### 1.4.2 语言模型简介

语言模型预测文本的概率分布，是NLP的核心技术。

**1. $n$-gram语言模型**
```
P(w₁, w₂, ..., wₙ) = Π P(wᵢ | w₁, w₂, ..., wᵢ₋₁)
基于历史$n$个词预测下一个词
```

**问题**：
- 数据稀疏问题
- 无法捕捉长距离依赖
- 词汇表大小受限

**2. RNN语言模型**
- 循环神经网络
- 序列建模
- 长期依赖问题

**3. LSTM语言模型**
- 长短期记忆网络
- 缓解梯度消失
- 更好的序列建模

### 1.4.3 词汇表与分词

**分词方法**：

1. **空格分词**
   - 简单，但中文不适用

2. **BPE (Byte Pair Encoding)**
   - 子词分词
   - 平衡词汇表大小和子词多样性
   - Transformer中使用

3. **WordPiece**
   - Google提出
   - 基于概率的字符级组合

4. **SentencePiece**
   - Google提出
   - 无需预先分词
   - 支持多种语言

**词汇表构建**：
- 统计词频
- 保留高频词
- 使用子词覆盖低频词
- 控制词汇表大小

## 代码示例

```python
# 简单的神经网络示例
import numpy as np

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def forward_pass(X, W1, W2):
    # 第一层
    hidden = sigmoid(np.dot(X, W1))
    # 输出层
    output = sigmoid(np.dot(hidden, W2))
    return output, hidden

def loss_function(y_true, y_pred):
    # 交叉熵损失
    epsilon = 1e-15
    y_pred = np.clip(y_pred, epsilon, 1 - epsilon)
    return -np.mean(y_true * np.log(y_pred) + (1 - y_true) * np.log(1 - y_pred))

# 简单的梯度计算示例
def compute_gradients(X, y_true, output, hidden, W1, W2):
    d_output = output - y_true  # 输出层误差

    d_hidden = np.dot(d_output, W2.T) * hidden * (1 - hidden)

    dW2 = np.dot(hidden.T, d_output)
    dW1 = np.dot(X.T, d_hidden)

    return dW1, dW2

# 训练循环
X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
y = np.array([[0], [1], [1], [0]])

W1 = np.random.randn(2, 4)
W2 = np.random.randn(4, 1)

learning_rate = 0.1
epochs = 1000

for epoch in range(epochs):
    output, hidden = forward_pass(X, W1, W2)
    loss = loss_function(y, output)
    dW1, dW2 = compute_gradients(X, y, output, hidden, W1, W2)

    W1 -= learning_rate * dW1
    W2 -= learning_rate * dW2

    if epoch % 100 == 0:
        print(f"Epoch {epoch}, Loss: {loss:.4f}")

print(f"Final output:\n{output}")
print(f"Final loss: {loss:.4f}")
```

```
Epoch 300, Loss: 0.6931

Epoch 900, Loss: 0.6931
Final output:

 [0.5004]]
Final loss: 0.6931
```
