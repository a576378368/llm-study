第四章：高级技术#

本章介绍强化学习与人类反馈（RLHF）、对比学习的高级应用、多任务学习以及模型压缩等大模型领域的进阶技术。

4.1 强化学习与大模型#

4.1.1 RLHF原理#

人类反馈强化学习（Reinforcement Learning from Human Feedback，RLHF）是训练对齐人类意图的大模型的关键技术。

RLHF三阶段流程：

阶段1: 预训练语言模型
        ↓
阶段2: 训练奖励模型 (Reward Model)
        ↓
阶段3: 使用PPO强化学习微调 (RL Fine-tuning)

为什么需要RLHF：

传统的监督学习存在以下问题：

  1. 泛化性差：模型可能只在训练数据分布内表现良好

  2. 缺乏人类偏好：无法捕捉人类的隐含偏好

  3. 安全性难以保证：可能产生有害或不当内容

RLHF通过引入人类反馈，让模型学习到更符合人类期望的行为。

4.1.2 Reward Modeling#

奖励模型用于学习人类的偏好。

训练数据收集：

class RewardModelDataset(torch.utils.data.Dataset):
    """奖励模型训练数据"""
    def __init__(self, prompt_response_pairs):
        """
        prompt_response_pairs: [(prompt, response_a, response_b, preference), ...]
        preference: 0表示A更好，1表示B更好
        """
        self.data = prompt_response_pairs

    def __len__(self):
        return len(self.data)

    def __getitem__(self, idx):
        prompt, response_a, response_b, preference = self.data[idx]
        return {
            'prompt': prompt,
            'response_a': response_a,
            'response_b': response_b,
            'preference': preference
        }

奖励模型架构：

class RewardModel(nn.Module):
    """奖励模型"""
    def __init__(self, base_model):
        super().__init__()
        self.base_model = base_model
        self.reward_head = nn.Linear(base_model.config.hidden_size, 1)

    def forward(self, input_ids, attention_mask):
        """
        输出每个位置的奖励分数
        通常使用最后一个token的表示来预测整体奖励
        """
        outputs = self.base_model(input_ids, attention_mask=attention_mask)

        # 使用最后一个token的表示
        last_hidden = outputs.last_hidden_state[:, -1, :]
        reward = self.reward_head(last_hidden)

        return reward.squeeze(-1)

    def compute_reward(self, prompt, response, tokenizer):
        """计算单个prompt-response对的奖励"""
        text = prompt + response
        encoding = tokenizer(text, return_tensors='pt', padding=True)

        reward = self.forward(encoding['input_ids'], encoding['attention_mask'])
        return reward

Bradley-Terry模型：

奖励模型基于Bradley-Terry理论，假设人类偏好服从 logistic 函数：

\[P(response_A \succ response_B | prompt) = \frac{1}{1 + \exp(-(r_A - r_B))}\]

损失函数：

def reward_model_loss(reward_a, reward_b, preference):
    """
    奖励模型损失
    reward_a, reward_b: 两个response的奖励分数
    preference: 0表示A好，1表示B好
    """
    # 奖励差异
    reward_diff = reward_a - reward_b

    # Log sigmoid损失
    # 如果preference=0（应该A好），则reward_diff应该为正
    # 如果preference=1（应该B好），则reward_diff应该为负
    label = 1 - preference  # 转换为: 0表示A好，1表示B好

    loss = -torch.log(torch.sigmoid(reward_diff) + 1e-8) * label - \
           torch.log(1 - torch.sigmoid(reward_diff) + 1e-8) * (1 - label)

    return loss.mean()

class RewardModelTrainer:
    def __init__(self, reward_model, learning_rate=1e-5):
        self.model = reward_model
        self.optimizer = torch.optim.AdamW(reward_model.parameters(), lr=learning_rate)

    def train_step(self, batch):
        prompt = batch['prompt']
        response_a = batch['response_a']
        response_b = batch['response_b']
        preference = batch['preference']

        # 编码prompt-response对
        encoding_a = self.tokenize(prompt, response_a)
        encoding_b = self.tokenize(prompt, response_b)

        # 计算奖励
        reward_a = self.model(encoding_a['input_ids'], encoding_a['attention_mask'])
        reward_b = self.model(encoding_b['input_ids'], encoding_b['attention_mask'])

        # 计算损失
        loss = reward_model_loss(reward_a, reward_b, preference)

        # 反向传播
        self.optimizer.zero_grad()
        loss.backward()
        self.optimizer.step()

        return loss.item()

4.1.3 PPO训练过程#

PPO（Proximal Policy Optimization）是RLHF中用于微调语言模型的核心算法。

PPO核心思想：

PPO通过限制策略更新的幅度来保证训练稳定：

\[L^{CLIP}(\theta) = \mathbb{E}[min(r_t(\theta) \hat{A}_t, clip(r_t(\theta), 1-\epsilon, 1+\epsilon) \hat{A}_t)]\]

其中 \(r_t(\theta) = \frac{\pi_\theta(a_t|s_t)}{\pi_{old}(a_t|s_t)}\) 是重要性采样比率。

语言模型PPO：

class PPOTrainer:
    """PPO训练器用于语言模型"""
    def __init__(self, model, ref_model, reward_model, config):
        self.model = model
        self.ref_model = ref_model  # 参考模型（防止偏移过大）
        self.reward_model = reward_model
        self.config = config

        # PPO参数
        self.clip_eps = config.clip_epsilon  # 通常为0.2
        self.kl_coef = config.kl_coef  # KL惩罚系数

    def compute_kl_divergence(self, log_probs, ref_log_probs):
        """计算KL散度"""
        return (log_probs - ref_log_probs).mean()

    def ppo_step(self, queries, responses, rewards):
        """
        执行一次PPO更新
        queries: 查询文本
        responses: 生成的响应
        rewards: 奖励分数
        """
        # 编码
        input_ids = self.tokenize(queries, responses)
        attention_mask = input_ids != self.tokenizer.pad_token_id

        # 前向传播
        outputs = self.model(input_ids, attention_mask=attention_mask)
        log_probs = outputs.log_probs  # 新策略的对数概率

        # 参考模型的对数概率
        with torch.no_grad():
            ref_outputs = self.ref_model(input_ids, attention_mask=attention_mask)
            ref_log_probs = ref_outputs.log_probs

        # 优势估计（简化版）
        advantages = rewards - rewards.mean()

        # 计算比率
        ratio = torch.exp(log_probs - ref_log_probs)

        # PPO裁剪损失
        surr1 = ratio * advantages
        surr2 = torch.clamp(ratio, 1 - self.clip_eps, 1 + self.clip_eps) * advantages
        policy_loss = -torch.min(surr1, surr2).mean()

        # KL惩罚
        kl_loss = self.compute_kl_divergence(log_probs, ref_log_probs)

        # 总损失
        total_loss = policy_loss + self.kl_coef * kl_loss

        # 更新
        self.optimizer.zero_grad()
        total_loss.backward()
        torch.nn.utils.clip_grad_norm_(self.model.parameters(), self.config.max_grad_norm)
        self.optimizer.step()

        return total_loss.item(), policy_loss.item(), kl_loss.item()

完整RLHF流程：

class RLHFTrainer:
    """完整的RLHF训练器"""
    def __init__(self, config):
        self.config = config

        # 加载预训练模型
        self.model = load_pretrained_model(config.model_name)
        self.ref_model = copy.deepcopy(self.model)  # 参考模型

        # 训练奖励模型
        print("训练奖励模型...")
        self.reward_model = self.train_reward_model()

        # PPO训练

        self.ppo_trainer = PPOTrainer(self.model, self.ref_model, self.reward_model, config)

    def train_reward_model(self):
        """训练奖励模型"""
        reward_model = RewardModel(copy.deepcopy(self.model))

        # 收集人类偏好数据
        preference_data = self.collect_preference_data()

        dataset = RewardModelDataset(preference_data)
        trainer = RewardModelTrainer(reward_model)

        for epoch in range(3):
            for batch in DataLoader(dataset, batch_size=8):
                trainer.train_step(batch)

        return reward_model

    def collect_preference_data(self):
        """收集人类偏好数据"""
        # 实际应用中需要人类标注
        # 这里示例数据格式
        return []

    def train(self, prompts):
        """执行完整的RLHF训练"""
        for iteration in range(self.config.num_iterations):
            print(f"\n=== 迭代 {iteration + 1} ===")

            for prompt in prompts:
                # 使用当前模型生成响应
                response = self.generate_response(prompt)

                # 计算奖励
                reward = self.reward_model.compute_reward(prompt, response, self.tokenizer)

                print(f"Reward: {reward:.4f}, Loss: {loss:.4f}")

            # 保存checkpoint
            self.save_checkpoint(iteration)

4.1.4 应用实例#

InstructGPT的RLHF流程：

  1. SFT阶段：使用人类编写的对话数据微调GPT-3

  2. 奖励模型阶段：训练模型预测人类偏好

  3. PPO阶段：使用PPO算法优化生成质量

class InstructGPTTraining:
    """InstructGPT风格的训练"""
    def __init__(self, base_model):
        self.model = base_model

    def stage1_supervised_finetuning(self, sft_data):
        """SFT阶段"""
        print("阶段1: 监督微调")
        # 使用人类标注的问答对进行微调
        for epoch in range(2):
            for batch in sft_data:
                outputs = self.model(**batch)
                loss = outputs.loss
                loss.backward()

    def stage2_reward_modeling(self, comparison_data):
        """奖励模型阶段"""
        print("阶段2: 训练奖励模型")
        reward_model = RewardModel(copy.deepcopy(self.model))

        # 训练数据: [prompt, response_a, response_b, preference]
        for batch in comparison_data:
            loss = self.compute_reward_loss(batch)
            loss.backward()

        return reward_model

    def stage3_rlhf(self, prompts, reward_model):
        """RLHF阶段"""
        print("阶段3: PPO强化学习")
        ppo_trainer = PPOTrainer(self.model, reward_model)

        for iteration in range(10):
            for prompt in prompts:
                response = self.model.generate(prompt)
                reward = reward_model.score(prompt, response)

                ppo_trainer.step(prompt, response, reward)

4.2 对比学习#

4.2.1 对比学习基础#

对比学习的核心是学习相似样本在表示空间中接近，不相似样本远离。

损失函数：

三元组损失（Triplet Loss）： $\(\mathcal{L} = \max(0, d(a, p) - d(a, n) + margin)\)$

其中 \(a\) 是锚点，\(p\) 是正样本，\(n\) 是负样本。

class TripletLoss(nn.Module):
    def __init__(self, margin=0.2):
        super().__init__()
        self.margin = margin

    def forward(self, anchor, positive, negative):
        dist_pos = torch.sum((anchor - positive) ** 2, dim=-1)
        dist_neg = torch.sum((anchor - negative) ** 2, dim=-1)
        loss = torch.relu(dist_pos - dist_neg + self.margin)
        return loss.mean()

InfoNCE损失（见前述章节）是最常用的对比学习损失。

4.2.2 SimCSE等模型#

SimCSE（Simple Contrastive Sentence Embedding）是一种高效的句子表示学习方法。

核心思想：

  1. 无监督SimCSE：使用dropout作为数据增强，同一句话两次通过模型得到正样本

  2. 有监督SimCSE：使用自然语言推理（NLI）数据集中的蕴含关系作为正样本

class SimCSE(nn.Module):
    """SimCSE模型"""
    def __init__(self, encoder):
        super().__init__()
        self.encoder = encoder

    def unsupervised_loss(self, input_ids, attention_mask):
        """
        无监督SimCSE损失
        正样本：同一文本，不同dropout
        负样本：批次内其他文本
        """
        # 两次前向传播（不同的dropout）
        outputs1 = self.encoder(input_ids, attention_mask, return_dict=True)
        outputs2 = self.encoder(input_ids, attention_mask, return_dict=True)

        # [CLS]表示
        z1 = outputs1.last_hidden_state[:, 0]
        z2 = outputs2.last_hidden_state[:, 0]

        # 归一化
        z1 = nn.functional.normalize(z1, dim=1)
        z2 = nn.functional.normalize(z2, dim=1)

        # 相似度矩阵
        similarity = torch.matmul(z1, z2.T)

        # 对角线为正样本
        batch_size = input_ids.size(0)
        labels = torch.arange(batch_size, device=input_ids.device)

        # InfoNCE
        loss = nn.CrossEntropyLoss()(similarity / 0.05, labels)

        return loss

    def supervised_loss(self, anchor, positive, negatives):
        """
        有监督SimCSE损失
        anchor: 锚文本
        positive: 正样本（蕴含）
        negatives: 负样本（矛盾）
        """
        anchor_emb = self.encoder(anchor).last_hidden_state[:, 0]
        positive_emb = self.encoder(positive).last_hidden_state[:, 0]

        # 多个负样本
        negative_embs = []
        for neg in negatives:
            neg_emb = self.encoder(neg).last_hidden_state[:, 0]
            negative_embs.append(neg_emb)

        # 计算损失
        all_emb = torch.cat([positive_emb.unsqueeze(1), torch.stack(negative_embs, dim=1)], dim=1)
        all_emb = nn.functional.normalize(all_emb, dim=-1)

        # 正样本对相似度
        pos_sim = torch.sum(anchor_emb * positive_emb, dim=-1, keepdim=True)

        # 与负样本的相似度
        neg_sim = torch.matmul(anchor_emb.unsqueeze(1), all_emb.transpose(-2, -1)).squeeze(1)

        # Softmax损失
        logits = torch.cat([pos_sim, neg_sim], dim=1) / 0.05
        labels = torch.zeros(logits.size(0), dtype=torch.long, device=logits.device)

        return nn.CrossEntropyLoss()(logits, labels)

4.2.3 对比预训练与微调#

对比预训练阶段：

class ContrastivePretrainingPipeline:
    """对比预训练流程"""
    def __init__(self, model, config):
        self.model = model
        self.config = config
        self.temperature = config.temperature or 0.05

    def contrastive_pretrain(self, texts):
        """
        对比预训练
        texts: 大量无标签文本
        """
        optimizer = torch.optim.AdamW(self.model.parameters(), lr=1e-4)

        for epoch in range(self.config.num_epochs):
            self.model.train()
            total_loss = 0

            for batch in self.create_batches(texts, batch_size=64):
                # 数据增强：回译、裁剪、同义词替换等
                view1 = self.augment(batch)
                view2 = self.augment(batch)

                # 编码
                emb1 = self.model(**self.tokenize(view1)).last_hidden_state[:, 0]
                emb2 = self.model(**self.tokenize(view2)).last_hidden_state[:, 0]

                # 归一化
                emb1 = nn.functional.normalize(emb1, dim=1)
                emb2 = nn.functional.normalize(emb2, dim=1)

                # InfoNCE损失
                similarity = torch.matmul(emb1, emb2.T) / self.temperature
                labels = torch.arange(len(batch), device=emb1.device)
                loss = nn.CrossEntropyLoss()(similarity, labels)

                optimizer.zero_grad()
                loss.backward()
                optimizer.step()

                total_loss += loss.item()

            print(f"Epoch {epoch}, Loss: {total_loss:.4f}")

    def augment(self, texts):
        """数据增强"""
        augmented = []
        for text in texts:
            # 随机选择增强方式
            if random.random() < 0.5:
                aug_text = self.back_translate(text)  # 回译
            else:
                aug_text = self.dropout_noise(text)  # dropout噪声
            augmented.append(aug_text)
        return augmented

对比微调阶段：

class ContrastiveFine-tuning:
    """对比微调"""
    def __init__(self, model):
        self.model = model

    def fine_tune_on_pairs(self, pairs, batch_size=32):
        """
        使用句子对进行对比微调
        pairs: [(sentence1, sentence2, label), ...] label=1表示相似，0表示不相似
        """
        optimizer = torch.optim.AdamW(self.model.parameters(), lr=2e-5)

        for epoch in range(3):
            random.shuffle(pairs)
            total_loss = 0

            for i in range(0, len(pairs), batch_size):
                batch = pairs[i:i+batch_size]

                sent1 = [p[0] for p in batch]
                sent2 = [p[1] for p in batch]
                labels = torch.tensor([p[2] for p in batch], dtype=torch.float32)

                # 编码
                emb1 = self.model(**self.tokenize(sent1)).last_hidden_state[:, 0]
                emb2 = self.model(**self.tokenize(sent2)).last_hidden_state[:, 0]

                emb1 = nn.functional.normalize(emb1, dim=1)
                emb2 = nn.functional.normalize(emb2, dim=1)

                # 余弦相似度
                similarity = torch.sum(emb1 * emb2, dim=-1)

                # 对比损失
                loss = nn.BCEWithLogitsLoss()(similarity, labels)

                optimizer.zero_grad()
                loss.backward()
                optimizer.step()

                total_loss += loss.item()

            print(f"Epoch {epoch}, Loss: {total_loss:.4f}")

4.3 多任务学习#

4.3.1 多任务预训练#

多任务预训练让模型同时学习多个任务，从而获得更广泛的能力。

多任务学习框架：

class MultiTaskModel(nn.Module):
    """多任务模型"""
    def __init__(self, encoder, task_heads):
        super().__init__()
        self.encoder = encoder
        self.task_heads = nn.ModuleDict(task_heads)  # 任务名称 -> 任务头

    def forward(self, input_ids, attention_mask, task_name):
        """
        多任务前向传播
        task_name: 当前任务名称
        """
        # 共享编码器
        encoder_outputs = self.encoder(input_ids, attention_mask=attention_mask)

        # 任务特定头
        task_head = self.task_heads[task_name]
        outputs = task_head(encoder_outputs)

        return outputs

class MultiTaskDataset(torch.utils.data.Dataset):
    """多任务数据集"""
    def __init__(self, task_datasets):
        """
        task_datasets: {task_name: dataset}
        """
        self.task_datasets = task_datasets
        self.task_names = list(task_datasets.keys())
        self.lengths = {name: len(ds) for name, ds in task_datasets.items()}

    def __len__(self):
        return sum(self.lengths.values())

    def __getitem__(self, idx):
        # 根据索引确定任务
        cumulative = 0
        for task_name, length in self.lengths.items():
            if idx < cumulative + length:
                data = self.task_datasets[task_name][idx - cumulative]
                data['task_name'] = task_name
                return data
            cumulative += length

class MultiTaskTrainer:
    """多任务训练器"""
    def __init__(self, model, tasks, task_weights=None):
        self.model = model
        self.tasks = tasks
        self.task_weights = task_weights or {t: 1.0 for t in tasks}

    def train(self, multi_task_dataset, num_epochs):
        """多任务训练"""
        dataloader = DataLoader(multi_task_dataset, batch_size=32, shuffle=True)

        for epoch in range(num_epochs):
            for batch in dataloader:
                task_name = batch.pop('task_name')

                # 前向传播
                outputs = self.model(
                    input_ids=batch['input_ids'],
                    attention_mask=batch['attention_mask'],
                    task_name=task_name
                )

                # 计算任务损失
                task_loss = self.compute_task_loss(outputs, batch, task_name)

                # 加权总损失
                total_loss = sum(
                    self.task_weights[t] * task_loss[t]
                    for t in task_loss.keys()
                )

                # 反向传播
                total_loss.backward()
                self.optimizer.step()
                self.optimizer.zero_grad()

专家混合模型（Mixture of Experts）：

class MoETransformer(nn.Module):
    """Mixture of Experts Transformer"""
    def __init__(self, d_model, num_experts=8, top_k=2):
        super().__init__()
        self.num_experts = num_experts
        self.top_k = top_k

        # 专家网络
        self.experts = nn.ModuleList([
            nn.Sequential(
                nn.Linear(d_model, d_model * 4),
                nn.GELU(),
                nn.Linear(d_model * 4, d_model)
            )
            for _ in range(num_experts)
        ])

        # 路由网络
        self.router = nn.Linear(d_model, num_experts)

    def forward(self, x):
        """
        x: (batch, seq_len, d_model)
        """
        batch_size, seq_len, d_model = x.shape

        # 路由计算
        router_logits = self.router(x)  # (batch, seq_len, num_experts)
        router_probs = F.softmax(router_logits, dim=-1)

        # 选择top-k专家
        top_k_probs, top_k_indices = torch.topk(router_probs, self.top_k, dim=-1)
        top_k_probs = top_k_probs / top_k_probs.sum(dim=-1, keepdim=True)  # 归一化

        # 计算输出
        output = torch.zeros_like(x)
        for k in range(self.top_k):
            expert_idx = top_k_indices[:, :, k]  # (batch, seq_len)
            expert_weight = top_k_probs[:, :, k].unsqueeze(-1)  # (batch, seq_len, 1)

            # 收集输入到对应专家
            for i in range(self.num_experts):
                mask = (expert_idx == i)  # (batch, seq_len)
                if mask.any():
                    expert_input = x[mask]
                    expert_output = self.experts[i](expert_input)
                    output[mask] += expert_output * expert_weight[mask]

        return output

渐进式迁移学习：

class ProgressiveTransfer:
    """渐进式跨领域迁移"""
    def __init__(self, model):
        self.model = model

    def transfer_to_target_domain(self, source_data, target_data, epochs_per_domain=3):
        """
        渐进式迁移
        从相近领域逐渐迁移到目标领域
        """
        # 按相似度排序领域
        domain_similarity = self.compute_domain_similarity(source_data, target_data)
        ordered_domains = sorted(domain_similarity.items(), key=lambda x: x[1], reverse=True)

        current_model = copy.deepcopy(self.model)

        for domain_name, similarity in ordered_domains:
            print(f"迁移到领域: {domain_name} (相似度: {similarity:.4f})")

            # 在该领域数据上微调
            domain_data = self.get_domain_data(domain_name)
            current_model = self.fine_tune(current_model, domain_data, epochs=epochs_per_domain)

            # 评估
            if self.evaluate(current_model, target_data) > 0.9:
                break

        return current_model

4.4 模型压缩#

4.4.1 知识蒸馏#

知识蒸馏将大模型的知识迁移到小模型。

蒸馏损失：

\[\mathcal{L}_{KD} = \alpha \cdot \mathcal{L}_{CE} + (1 - \alpha) \cdot \mathcal{L}_{soft}\]

其中 \(\mathcal{L}_{soft}\) 是软目标的知识蒸馏损失：

\[\mathcal{L}_{soft} = - \sum_j p_T(s_j) \log p_T(t_j)\]

\(p_T(s_j)\) 是教师模型的概率输出（带温度\(T\)），\(p_T(t_j)\) 是学生模型的概率输出。

class KnowledgeDistillation:
    """知识蒸馏"""
    def __init__(self, teacher_model, student_model, temperature=4.0, alpha=0.5):
        self.teacher = teacher_model
        self.student = student_model
        self.temperature = temperature
        self.alpha = alpha

    def distillation_loss(self, teacher_logits, student_logits, labels):
        """
        计算蒸馏损失
        teacher_logits: 教师模型输出 (logits)
        student_logits: 学生模型输出 (logits)
        """
        # 软损失：知识蒸馏损失
        soft_teacher = F.softmax(teacher_logits / self.temperature, dim=-1)
        soft_student = F.log_softmax(student_logits / self.temperature, dim=-1)
        loss_soft = -torch.sum(soft_teacher * soft_student, dim=-1).mean()

        # 硬损失：常规交叉熵
        loss_hard = F.cross_entropy(student_logits, labels)

        # 加权组合
        loss = self.alpha * (self.temperature ** 2) * loss_soft + (1 - self.alpha) * loss_hard

        return loss

    def train(self, train_loader):
        """蒸馏训练"""
        optimizer = torch.optim.AdamW(self.student.parameters(), lr=1e-4)

        for epoch in range(5):
            for batch in train_loader:
                input_ids = batch['input_ids']
                attention_mask = batch['attention_mask']
                labels = batch['labels']

                # 教师模型前向（不更新）
                with torch.no_grad():
                    teacher_logits = self.teacher(input_ids, attention_mask).logits

                # 学生模型前向
                student_logits = self.student(input_ids, attention_mask).logits

                # 计算蒸馏损失
                loss = self.distillation_loss(teacher_logits, student_logits, labels)

                optimizer.zero_grad()
                loss.backward()
                optimizer.step()

                print(f"Epoch {epoch}, Loss: {loss.item():.4f}")

中间层蒸馏：

class IntermediateLayerDistillation:
    """中间层蒸馏"""
    def __init__(self, teacher, student):
        self.teacher = teacher
        self.student = student

    def distill_hidden_states(self, teacher_hidden, student_hidden, layer_mapping):
        """
        蒸馏中间层表示
        layer_mapping: 教师层到学生层的映射
        """
        loss = 0
        for teacher_layer, student_layer in layer_mapping:
            t_hidden = teacher_hidden[teacher_layer]
            s_hidden = student_hidden[student_layer]

            # MSE损失
            loss += F.mse_loss(s_hidden, t_hidden)

        return loss

4.4.2 量化技术#

量化通过降低权重精度来减少模型大小和加速推理。

训练后量化（PTQ）：

class PostTrainingQuantization:
    """训练后量化"""
    def __init__(self, model, bits=8):
        self.model = model
        self.bits = bits

    def dynamic_quantization(self):
        """动态量化（权重动态量化）"""
        quantized_model = torch.quantization.quantize_dynamic(
            self.model,
            {nn.Linear, nn.LSTM, nn.MultiheadAttention},
            dtype=torch.qint8
        )
        return quantized_model

    def static_quantization(self, calibration_data):
        """静态量化（需要校准数据）"""
        # 模型转换
        self.model.eval()
        self.model.qconfig = torch.quantization.get_default_qconfig('fbgemm')
        torch.quantization.prepare(self.model, inplace=True)

        # 校准
        print("校准中...")
        with torch.no_grad():
            for batch in calibration_data:
                self.model(**batch)

        # 转换
        quantized_model = torch.quantization.convert(self.model, inplace=False)
        return quantized_model

    def int4_quantization(self, model):
        """INT4量化（使用GPTQ或AWQ）"""
        # 权重分块量化
        def quantize_weight(weight, bits=4):
            # 计算量化参数
            scale = (weight.max() - weight.min()) / (2 ** bits - 1)
            zero_point = -weight.min() / scale

            # 量化
            quantized = torch.round(weight / scale + zero_point)
            quantized = torch.clamp(quantized, 0, 2 ** bits - 1)

            return quantized.to(torch.uint8), scale, zero_point

        # 应用量化
        for name, module in model.named_modules():
            if isinstance(module, nn.Linear):
                quantized_weight, scale, zero_point = quantize_weight(module.weight.data)
                module.weight.data = dequantize(quantized_weight, scale, zero_point)

        return model

量化感知训练（QAT）：

class QuantizationAwareTraining:
    """量化感知训练"""
    def __init__(self, model):
        self.model = model

    def prepare_qat(self):
        """准备QAT"""
        self.model.train()
        self.model.qconfig = torch.quantization.QConfig(
            activation=torch.quantization.Observer.default_with_subsample,
            weight=torch.quantization.default_weight_observer_range
        )
        torch.quantization.prepare_qat(self.model, inplace=True)

    def train_qat(self, train_loader, epochs=3):
        """量化感知训练"""
        optimizer = torch.optim.AdamW(self.model.parameters(), lr=1e-4)

        for epoch in range(epochs):
            for batch in train_loader:
                outputs = self.model(**batch)
                loss = outputs.loss

                optimizer.zero_grad()
                loss.backward()
                optimizer.step()

        return self.model

    def convert_to_quantized(self):
        """转换到量化模型"""
        self.model.eval()
        quantized_model = torch.quantization.convert(self.model, inplace=False)
        return quantized_model

4.4.3 剪枝方法#

剪枝移除不重要的权重或神经元。

结构化剪枝：

class StructuredPruning:
    """结构化剪枝"""
    def __init__(self, model, sparsity=0.5):
        self.model = model
        self.sparsity = sparsity

    def magnitude_pruning(self):
        """幅度剪枝"""
        for name, module in self.model.named_modules():
            if isinstance(module, nn.Linear):
                # 计算权重幅度
                weight magnitude = torch.abs(module.weight.data)

                # 确定阈值
                threshold = torch.quantile(weight_magnitude, self.sparsity)

                # 创建掩码
                mask = weight_magnitude > threshold
                module.weight.data *= mask.float()

    def gradient_pruning(self):
        """梯度重要性剪枝"""
        for name, module in self.model.named_modules():
            if isinstance(module, nn.Linear):
                # 计算梯度幅度
                grad_magnitude = torch.abs(module.weight.grad)

                threshold = torch.quantile(grad_magnitude, self.sparsity)
                mask = grad_magnitude > threshold

                module.weight.grad *= mask.float()

    def lottery_ticket_pruning(self, train_loader, iterations=10, sparsity_schedule=None):
        """
        彩票票据剪枝
        迭代式剪枝和重训练
        """
        self.model.train()

        for iteration in range(iterations):
            # 当前稀疏度
            current_sparsity = sparsity_schedule(iteration) if sparsity_schedule else self.sparsity

            print(f"迭代 {iteration + 1}, 稀疏度: {current_sparsity:.2%}")

            # 训练一轮
            for batch in train_loader:
                outputs = self.model(**batch)
                loss = outputs.loss
                loss.backward()
                self.optimizer.step()
                self.optimizer.zero_grad()

            # 剪枝
            self.prune_by_sparsity(current_sparsity)

            # 重置到初始权重（彩票票据假设）
            self.reset_weights()

        return self.model

    def prune_by_sparsity(self, target_sparsity):
        """按目标稀疏度剪枝"""
        for name, module in self.model.named_modules():
            if isinstance(module, nn.Linear):
                weight = module.weight.data
                num_params = weight.numel()
                num_to_keep = int(num_params * (1 - target_sparsity))

                # 保留最大的权重
                _, sorted_indices = torch.sort(torch.abs(weight).flatten(), descending=True)
                mask = torch.zeros_like(weight, dtype=torch.bool)
                mask.flatten()[sorted_indices[:num_to_keep]] = True

                module.weight.data *= mask.float()

非结构化剪枝：

class UnstructuredPruning:
    """非结构化剪枝（细粒度剪枝）"""
    def __init__(self, model):
        self.model = model

    def magnitude_based_pruning(self, sparsity=0.5):
        """基于幅度的非结构化剪枝"""
        for name, module in self.model.named_modules():
            if isinstance(module, nn.Linear):
                weight = module.weight.data
                threshold = torch.quantile(torch.abs(weight), sparsity)

                mask = torch.abs(weight) > threshold
                module.weight.data *= mask.float()

                print(f"Layer {name}: {mask.sum()}/{mask.numel()} 保留")

    def sparse_model_export(self):
        """导出稀疏模型（使用稀疏格式）"""
        # 将密集权重转为稀疏表示
        sparse_weights = {}

        for name, module in self.model.named_modules():
            if isinstance(module, nn.Linear):
                weight = module.weight.data
                mask = weight != 0

                # COOrdinate格式
                sparse_weights[name] = {
                    'indices': mask.nonzero(as_tuple=False),
                    'values': weight[mask],
                    'shape': weight.shape
                }

        return sparse_weights
