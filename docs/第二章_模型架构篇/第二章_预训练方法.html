第二章：预训练方法#

预训练是大模型获得强大能力的关键环节。本章详细介绍自监督学习任务、数据预处理、训练策略等内容。

2.1 自监督学习任务#

自监督学习利用数据自身的结构作为监督信号，无需人工标注。预训练任务的设计直接决定了模型学习到的表示质量。

2.1.1 掩码语言建模（MLM）#

掩码语言建模是BERT等模型使用的核心预训练任务。

原理：

输入文本: "今天天气很好"
掩码后:   "今天 [MASK] 很好"
目标:     预测被掩码的词 "天气"

数学表述：

给定一个序列 \(x = (x_1, x_2, ..., x_n)\)，随机选择一部分位置进行掩码。设 \(\mathcal{M}\) 表示被掩码的位置集合，掩码比例为 \(15\%\)。

模型学习最大化以下对数似然：

\[\mathcal{L}_{MLM} = \sum_{i \in \mathcal{M}} \log P(x_i | x_{\setminus \mathcal{M}})\]

掩码策略：

  1. 随机掩码：以固定概率随机选择token进行掩码

  2. 词边界掩码：对完整词或短语进行掩码（如ERNIE）

  3. 动态掩码：每次训练时动态生成不同的掩码模式

代码实现：

import torch
import torch.nn as nn
import random

class MaskedLanguageModeling:
    """掩码语言建模数据处理"""
    def __init__(self, tokenizer, mask_prob=0.15):
        self.tokenizer = tokenizer
        self.mask_prob = mask_prob

    def mask_tokens(self, input_ids, attention_mask):
        """
        对输入序列进行随机掩码
        input_ids: (batch_size, seq_len)
        返回: masked_input_ids, labels
        """
        labels = input_ids.clone()
        probability_matrix = torch.full(input_ids.shape, self.mask_prob)

        # 不对特殊token进行掩码（[CLS], [SEP], [PAD]等）
        special_tokens_mask = self.tokenizer.get_special_tokens_mask(input_ids)
        probability_matrix.masked_fill_(
            torch.tensor(special_tokens_mask, dtype=torch.bool),
            value=0.0
        )

        masked_indices = torch.bernoulli(probability_matrix).bool()
        labels[~masked_indices] = -100  # 非掩码位置不计算损失

        # 80%替换为[MASK]，10%替换为随机词，10%保持不变
        indices_replaced = torch.bernoulli(torch.full(labels.shape, 0.8)).bool() & masked_indices
        input_ids[indices_replaced] = self.tokenizer.mask_token_id

        indices_random = torch.bernoulli(torch.full(labels.shape, 0.5)).bool() & masked_indices & ~indices_replaced
        random_words = torch.randint(len(self.tokenizer), labels.shape, dtype=torch.long)
        input_ids[indices_random] = random_words[indices_random]

        return input_ids, labels

2.1.2 下一句预测（NSP）#

下一句预测任务帮助模型理解句子间的关系。

原理：

训练数据由句子对组成：

  • 正样本：实际相邻的句子对

  • 负样本：随机组合的句子对

模型判断第二个句子是否紧跟第一个句子。

数学表述：

给定句子对 \((s_A, s_B)\)，预测标签 \(y \in \{0, 1\}\)：

  • \(y = 1\)：\(s_B\) 是 \(s_A\) 的下一个句子

  • \(y = 0\)：\(s_B\) 不是 \(s_A\) 的下一个句子

\[\mathcal{L}_{NSP} = -[y \log P(next) + (1-y) \log P(not\_next)]\]

数据构造：

import random

def create_nsp_data(document, tokenizer, max_length=512):
    """
    从文档中构造NSP训练数据
    document: 句子列表
    """
    examples = []
    for i in range(len(document) - 1):
        is_next = 1  # 正样本：下一个句子

        # 50%概率构造负样本
        if random.random() < 0.5:
            j = random.randint(0, len(document) - 1)
            while j == i:
                j = random.randint(0, len(document) - 1)
            sentence_b = document[j]
            is_next = 0  # 负样本
        else:
            sentence_b = document[i + 1]

        # 拼接并编码
        encoding = tokenizer(
            sentence_a,
            sentence_b,
            max_length=max_length,
            padding='max_length',
            truncation=True,
            return_tensors='pt'
        )

        examples.append({
            'input_ids': encoding['input_ids'],
            'attention_mask': encoding['attention_mask'],
            'labels': is_next
        })

    return examples

NSP的争议：

后续研究表明NSP任务对模型性能的提升有限：

  • RoBERTa发现去除NSP后性能反而提升

  • ALBERT通过Sentence Order Prediction（SOP）替代NSP

  • 现代模型更多关注MLM的改进

2.1.3 填空任务#

填空任务（Gap Sentence Prediction，GSP）是对MLM的增强。

种类：

  1. 句序预测（SOP）：判断两个句子的正确顺序

  2. 句子顺序预测：给定多个句子，判断排列顺序

  3. 段落重排序：打乱段落顺序，预测正确顺序

Sentence Order Prediction (SOP)：

class SentenceOrderPrediction:
    """ALBERT使用的SOP任务"""
    def __init__(self, tokenizer):
        self.tokenizer = tokenizer

    def create_sop_example(self, sentence_a, sentence_b):
        """
        构造SOP训练样本
        正样本：原始顺序 (A, B)
        负样本：交换顺序 (B, A)
        """
        # 正样本
        positive_encoding = self.tokenizer(
            sentence_a, sentence_b,
            max_length=512,
            padding='max_length',
            truncation=True,
            return_tensors='pt'
        )

        # 负样本（交换顺序）
        negative_encoding = self.tokenizer(
            sentence_b, sentence_a,
            max_length=512,
            padding='max_length',
            truncation=True,
            return_tensors='pt'
        )

        return positive_encoding, negative_encoding

2.1.4 对比学习任务#

对比学习通过比较正负样本学习表示。

InfoNCE损失：

给定查询 \(q\) 和键值对 \((k^+, k^-)\)，对比学习最大化 \(q\) 与 \(k^+\) 的相似度，同时最小化与 \(k^-\) 的相似度。

\[\mathcal{L}_{对比} = -\log \frac{\exp(q \cdot k^+ / \tau)}{\sum_{i=0}^{N} \exp(q \cdot k_i / \tau)}\]

其中 \(\tau\) 是温度参数。

SimCSE对比学习：

class SimCSEPretraining:
    """SimCSE对比学习预训练"""
    def __init__(self, model, temperature=0.05):
        self.model = model
        self.temperature = temperature

    def simcse_loss(self, input_ids, attention_mask):
        """
        SimCSE的对比学习损失
        正样本：同一句子两次dropout的不同输出
        负样本：批次内其他句子
        """
        # 前向传播两次，通过不同的dropout
        outputs1 = self.model(input_ids, attention_mask)
        outputs2 = self.model(input_ids, attention_mask)

        # 取[CLS]token的表示
        z1 = outputs1.last_hidden_state[:, 0]  # (batch_size, hidden)
        z2 = outputs2.last_hidden_state[:, 0]

        # 计算相似度
        z1 = torch.nn.functional.normalize(z1, dim=1)
        z2 = torch.nn.functional.normalize(z2, dim=1)

        # 相似度矩阵
        sim_matrix = torch.matmul(z1, z2.T) / self.temperature  # (batch, batch)

        # 对角线为正样本
        labels = torch.arange(len(input_ids), device=input_ids.device)

        loss = torch.nn.functional.cross_entropy(sim_matrix, labels)
        return loss

数据清洗流程：

import re
from collections import Counter

class DataCleaner:
    """数据清洗工具"""
    def __init__(self, min_length=10, max_length=4096):
        self.min_length = min_length
        self.max_length = max_length

    def clean_text(self, text):
        """清洗单个文本"""
        # 移除特殊控制字符
        text = re.sub(r'[\x00-\x08\x0b-\x0c\x0e-\x1f\x7f-\x9f]', '', text)

        # 规范化Unicode
        text = text.strip()

        # 移除过长或过短的文本
        if len(text) < self.min_length or len(text) > self.max_length:
            return None

        return text

    def remove_duplicates(self, texts, hash_func=None):
        """去除重复文本"""
        if hash_func is None:
            hash_func = lambda x: hash(x)

        seen = set()
        unique_texts = []

        for text in texts:
            h = hash_func(text)
            if h not in seen:
                seen.add(h)
                unique_texts.append(text)

        return unique_texts

    def filter_by_quality(self, texts, language='zh'):
        """基于质量过滤文本"""
        # 语言检测
        valid_texts = []
        for text in texts:
            # 简单启发式：检查中文字符比例
            if language == 'zh':
                chinese_chars = len(re.findall(r'[一-鿿]', text))
                if chinese_chars / len(text) < 0.3:  # 中文字符不足30%
                    continue
            valid_texts.append(text)

        return valid_texts

去重策略：

  1. 句子级去重：移除完全相同的句子

  2. 段落级去重：使用n-gram指纹

  3. 文档级去重：MinHash或SimHash

from datasketch import MinHash

def minhash_deduplication(texts, threshold=0.8):
    """
    使用MinHash进行近似去重
    threshold: 相似度阈值
    """
    def get_minhash(text):
        m = MinHash(num_perm=128)
        words = text.split()
        for d in words:
            m.update(d.encode('utf8'))
        return m

    duplicates = []
    signatures = []

    for text in texts:
        m = get_minhash(text)
        is_duplicate = False

        for i, existing_sig in enumerate(signatures):
            # 计算Jaccard相似度
            similarity = m.jaccard(existing_sig)
            if similarity > threshold:
                duplicates.append(i)
                is_duplicate = True
                break

        if not is_duplicate:
            signatures.append(m)

    return [text for i, text in enumerate(texts) if i not in duplicates]

2.2.2 分词方法#

分词是将文本转换为token序列的过程。

词级分词：

"今天天气很好" → ["今天", "天气", "很好"]

子词级分词：

"今天天气很好" → ["今天", "天气", "很", "好"]

BPE（Byte-Pair Encoding）：

class BPETokenizer:
    """BPE分词器实现"""
    def __init__(self, vocab_size=30000):
        self.vocab_size = vocab_size
        self.merges = []
        self.vocab = {}

    def get_stats(self, words):
        """统计pair频率"""
        pairs = Counter()
        for word, freq in words.items():
            symbols = word.split()
            for i in range(len(symbols) - 1):
                pairs[(symbols[i], symbols[i+1])] += freq
        return pairs

    def merge_vocab(self, pair, vocab):
        """合并最常见的pair"""
        out_vocab = {}
        bigram = ' '.join(pair)
        replacement = ''.join(pair)
        for word in vocab:
            w_out = word.replace(bigram, replacement)
            out_vocab[w_out] = vocab[word]
        return out_vocab

    def train(self, texts, min_freq=2):
        """训练BPE分词器"""
        # 初始化词表（字符级）
        words = Counter()
        for text in texts:
            word = ' '.join(list(text)) + ' </w>'
            words[word] += 1

        self.vocab = {chr(i): i for i in range(256)}
        self.vocab['</w>'] = 256

        # 迭代合并
        while len(self.vocab) < self.vocab_size:
            pairs = self.get_stats(words)
            if not pairs:
                break

            best_pair = max(pairs, key=pairs.get)
            if pairs[best_pair] < min_freq:
                break

            words = self.merge_vocab(best_pair, words)
            self.merges.append(best_pair)

            # 更新词表
            new_token = ''.join(best_pair)
            self.vocab[new_token] = len(self.vocab)

    def tokenize(self, text):
        """分词"""
        # 字符级分词
        tokens = list(text) + ['</w>']

        # 应用合并规则
        for merge in self.merges:
            new_tokens = []
            i = 0
            while i < len(tokens):
                if i < len(tokens) - 1 and (tokens[i], tokens[i+1]) == merge:
                    new_tokens.append(''.join(merge))
                    i += 2
                else:
                    new_tokens.append(tokens[i])
                    i += 1
            tokens = new_tokens

        return tokens

import random

def random_token_replacement(input_ids, mask_token_id, replace_prob=0.15):
    """随机替换token"""
    labels = input_ids.clone()
    labels[random.random() > replace_prob] = -100

    # 随机替换
    for i in range(len(input_ids)):
        if labels[i] != -100 and random.random() < replace_prob:
            if random.random() < 0.8:
                input_ids[i] = mask_token_id  # 替换为[MASK]
            elif random.random() < 0.5:
                input_ids[i] = random.randint(0, 30000)  # 随机词

    return input_ids, labels

from synonyms import similar

def synonym_replacement(text, n=3):
    """同义词替换"""
    words = text.split()
    for _ in range(n):
        idx = random.randint(0, len(words) - 1)
        similar_words = similar(words[idx])
        if similar_words:
            words[idx] = random.choice(similar_words)
    return ' '.join(words)

2.3 训练策略#

2.3.1 预训练流程设计#

训练流程概览：

数据准备 → Tokenization → Batching → 预训练 → Checkpoint保存

完整训练流程：

import torch
from torch.utils.data import DataLoader
from torch.optim import AdamW
from torch.cuda.amp import autocast, GradScaler

class PretrainingPipeline:
    def __init__(self, model, config):
        self.model = model
        self.config = config
        self.device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')

    def train(self, train_dataset, num_epochs, batch_size, learning_rate):
        # 数据加载
        dataloader = DataLoader(
            train_dataset,
            batch_size=batch_size,
            shuffle=True,
            num_workers=4,
            pin_memory=True
        )

        # 优化器
        optimizer = AdamW(self.model.parameters(), lr=learning_rate)

        # 学习率调度器
        scheduler = torch.optim.lr_scheduler.CosineAnnealingLR(
            optimizer, T_max=num_epochs * len(dataloader)
        )

        # 混合精度
        scaler = GradScaler()

        # 训练循环
        global_step = 0
        for epoch in range(num_epochs):
            self.model.train()
            epoch_loss = 0

            for step, batch in enumerate(dataloader):
                batch = {k: v.to(self.device) for k, v in batch.items()}

                optimizer.zero_grad()

                # 混合精度前向传播
                with autocast():
                    outputs = self.model(**batch)
                    loss = outputs.loss

                # 反向传播
                scaler.scale(loss).backward()
                scaler.unscale_(optimizer)
                torch.nn.utils.clip_grad_norm_(self.model.parameters(), 1.0)
                scaler.step(optimizer)
                scaler.update()

                scheduler.step()
                global_step += 1

                epoch_loss += loss.item()

                if step % 100 == 0:
                    print(f"Epoch {epoch}, Step {step}, Loss: {loss.item():.4f}")

            avg_loss = epoch_loss / len(dataloader)
            print(f"Epoch {epoch} completed, Average Loss: {avg_loss:.4f}")

            # 保存checkpoint
            if (epoch + 1) % 5 == 0:
                self.save_checkpoint(f"checkpoint_epoch_{epoch}.pt", global_step)

    def save_checkpoint(self, path, step):
        torch.save({
            'step': step,
            'model_state_dict': self.model.state_dict(),
        }, path)

def create_pipeline_schedule(model, num_stages, num_microbatches):
    """
    创建流水线并行调度
    将模型分为多个阶段，每个阶段在不同设备上执行
    """
    stages = torch.nn.Sequential(*list(model.children()))

    schedule = []
    for microbatch in range(num_microbatches):
        for stage_id, stage in enumerate(stages):
            schedule.append((stage_id, microbatch))

    return schedule

from torch.cuda.amp import autocast, GradScaler

class MixedPrecisionTrainer:
    def __init__(self, model):
        self.model = model
        self.scaler = GradScaler()

    def train_step(self, batch):
        optimizer = self.get_optimizer()

        # 前向传播使用自动混合精度
        with autocast(dtype=torch.bfloat16):
            outputs = self.model(**batch)
            loss = outputs.loss

        # 反向传播
        self.scaler.scale(loss).backward()

        # 梯度裁剪
        self.scaler.unscale_(optimizer)
        torch.nn.utils.clip_grad_norm_(self.model.parameters(), 1.0)

        # 更新参数
        self.scaler.step(optimizer)
        self.scaler.update()

        return loss

2.3.4 梯度累积#

梯度累积允许使用更大的有效批大小。

class GradientAccumulator:
    def __init__(self, model, accumulation_steps=4, batch_size=16):
        self.model = model
        self.accumulation_steps = accumulation_steps
        self.batch_size = batch_size
        self.optimizer = AdamW(model.parameters(), lr=1e-4)

    def train_step(self, batches):
        """累积多个小批次的梯度"""
        self.optimizer.zero_grad()

        for i, batch in enumerate(batches):
            outputs = self.model(**batch)
            loss = outputs.loss

            # 缩放损失以平均梯度
            loss = loss / self.accumulation_steps
            loss.backward()

            # 每accumulation_steps步更新一次
            if (i + 1) % self.accumulation_steps == 0:
                torch.nn.utils.clip_grad_norm_(self.model.parameters(), 1.0)
                self.optimizer.step()
                self.optimizer.zero_grad()

        return loss.item() * self.accumulation_steps

def initialize_bert(model):
    """BERT风格的初始化"""
    for name, param in model.named_parameters():
        if 'bias' in name:
            torch.nn.init.zeros_(param)
        elif 'LayerNorm' in name:
            torch.nn.init.zeros_(param)
            if 'weight' in name:
                torch.nn.init.ones_(param)
        elif 'weight' in name and 'embedding' not in name:
            torch.nn.init.normal_(param, mean=0.0, std=0.02)
        elif 'embedding' in name:
            torch.nn.init.normal_(param, mean=0.0, std=0.02)

def deepnorm_init(model, depth):
    """
    DeepNorm初始化
    适用于深层Transformer
    """
    std = (2 * depth) ** -0.5

    for name, param in model.named_parameters():
        if 'bias' not in name and 'LayerNorm' not in name:
            torch.nn.init.normal_(param, mean=0.0, std=std)

    return model

def create_optimizer_with_decay(model, lr, weight_decay=0.01):
    """带权重衰减的优化器"""
    decay_params = []
    no_decay_params = []

    for name, param in model.named_parameters():
        if not param.requires_grad:
            continue
        # 不对bias和LayerNorm应用权重衰减
        if 'bias' in name or 'LayerNorm' in name:
            no_decay_params.append(param)
        else:
            decay_params.append(param)

    optimizer_groups = [
        {'params': decay_params, 'weight_decay': weight_decay},
        {'params': no_decay_params, 'weight_decay': 0.0}
    ]

    return torch.optim.AdamW(optimizer_groups, lr=lr)

学习率预热：

class WarmupScheduler:
    """学习率预热调度器"""
    def __init__(self, optimizer, warmup_steps, total_steps):
        self.optimizer = optimizer
        self.warmup_steps = warmup_steps
        self.total_steps = total_steps

    def get_lr(self, step):
        if step < self.warmup_steps:
            return step / self.warmup_steps
        else:
            progress = (step - self.warmup_steps) / (self.total_steps - self.warmup_steps)
            return 0.5 * (1 + np.cos(np.pi * progress))

    def step(self, step):
        lr = self.get_lr(step)
        for i, group in enumerate(self.optimizer.param_groups):
            group['lr'] = lr * group.get('base_lr', 1.0)

class RegularizedTransformer(nn.Module):
    def __init__(self, base_model, dropout=0.1, label_smoothing=0.1):
        super().__init__()
        self.model = base_model
        self.dropout = nn.Dropout(dropout)
        self.label_smoothing = label_smoothing

    def forward(self, input_ids, attention_mask, labels):
        outputs = self.model(input_ids, attention_mask)

        # Label smoothing
        if labels is not None:
            loss_fct = nn.CrossEntropyLoss(label_smoothing=self.label_smoothing)
            logits = outputs.logits
            loss = loss_fct(
                logits.view(-1, logits.size(-1)),
                labels.view(-1)
            )
            return outputs

        return outputs

早停策略：

class EarlyStopping:
    def __init__(self, patience=5, min_delta=0.01):
        self.patience = patience
        self.min_delta = min_delta
        self.counter = 0
        self.best_loss = None
        self.early_stop = False

    def __call__(self, val_loss):
        if self.best_loss is None:
            self.best_loss = val_loss
        elif val_loss > self.best_loss - self.min_delta:
            self.counter += 1
            if self.counter >= self.patience:
                self.early_stop = True
        else:
            self.best_loss = val_loss
            self.counter = 0
